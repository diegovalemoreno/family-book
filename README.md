# Family-Book

Esse repositório faz parte de um estudo sobre React e ReactJS.

## Tecnologias e Conceitos

- React e ReactJS
- Webpack
- Babel
- Criação de Components

A aplicação é resultado do estudo de conceitos básicos da biblioteca.
